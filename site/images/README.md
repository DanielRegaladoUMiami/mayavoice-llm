## Images needed

Place these files in this folder:

- `logo.png` — MayaVoice logo (used in navbar). Recommended: 200x50px, transparent background.
- `favicon.png` — Browser tab icon. Recommended: 32x32px or 64x64px.

Optional (for future use):
- `hero-bg.jpg` — Hero background image
- `guatemala-map.png` — Map showing where languages are spoken
